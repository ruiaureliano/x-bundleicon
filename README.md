## Usage

```
USAGE: xbi [--version] <bundle> <size> <file>

    OPTIONS:
        -v, --version   Print the version and exit
        -h, --help      Show help information.

    SUBCOMMANDS:
        bundle          The app bundle id
        size            The icon size to export
        file            The file destination to save
